Первоначальная настройка (один раз)

  1. Зарегистрируйтесь на supabase.com (бесплатно)
  2. Создайте проект → SQL Editor → вставьте код из supabase/schema.sql
  3. Storage → New bucket → имя products → Public
  4. Authentication → Users → Add user (email + пароль для входа в админку)
  5. Settings → API → скопируйте URL и anon key в js/config.js
  6. Загрузите сайт на GitHub Pages заново
